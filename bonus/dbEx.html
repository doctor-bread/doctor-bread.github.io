The "db-statement" Attribute

  • "hover"
    Hovering over this span tag => db-if (Hovering) db-else (Not hovering) will switch it.
  • "mobile"
    This span tag changes when the screen is less than 768px. => db-if (Less than 768px) db-else
  • "checked"
    This span tag changes when the first input stated inside is checked, any checkbox input is ignored. I colored the useless inputs red for a more noticable difference. => db-if (first input checked) db-else (first input isn't checked)

The 'db-resize' Attribute

  • x, y & xy
    Defines how an element can be resized.

    db-resize="xy"

  • noresizer
    Hides the resizer.

    db-resize="xy noresizer"

  • noscroll
    Sets overflow to hidden.

    db-resize="xy noscroll"
